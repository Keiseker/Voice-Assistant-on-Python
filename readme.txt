Чтобы запустить голосового помощника на Python необходимо:
1. Распаковать архив в 1 папку.
2. Открыть эту папку или только файл main.py в вашей среде разработки.
3. В терминале запустить команду:
pip install -r requirements.txt
4. После установки всех зависимостей прописать в терминале команду:
python main.py
5. Далее кликнуть 1 раз на кнопку "Начать" и следовать инструкции на экране.
